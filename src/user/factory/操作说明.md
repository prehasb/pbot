# 矿石变量

## 在合成加工时矿石对锅炉的影响

矿石权重priviority。0~100。
在无合成表时决定合成什么东西

矿石对温度的影响add_tempurature。0~100。
在投入矿石后改变温度

矿石熔点melting_point。-273~2000。决定矿石合成的方式

## 矿石的固有属性

延展性ductility。0~100。

美观beauty。0~100。

硬度hardness。0~100。

密度density。0~正常值。

## 其他属性

描述description，对矿石的详细描述。

# 操作方法

初始化：空锅

操作：

丢入物品，如矿石，燃料等。从stone一列检测是否具有该物品。

具体操作：

- 根据活性、温度、权重、合成表进行反应

后续：
- 改变温度
- 改变活性，大于100时爆炸并扣除水晶，产生废料

取出物品：将物品存储在minedata文件中的stone一列。

数据格式：

水晶+3|水晶聚合物+5|水晶聚合物+5|燃料+3|废料|废料|废料|水晶聚合物+5|废料|

每个矿石都是独一无二的


# 文件格式

- minedata.csv

```csv
user_id,stonename,level,tempurature,activity,ductility,beauty,hardness,density,id
```

- stone_table.csv

```csv
id,name,english_name,priviority,add_tempurature,melting_point,add_activity,ductility,beauty,hardness,density,description
1,无,none,None,None,None,None,None,None,None,None,空空如也
2,水晶,crystal,2,-2,500,5,1,10,70,2.65,普通的水晶
3,水晶聚合物,crystal_cluster,3,-2,500,5,1,30,70,2.65,水晶的聚合物，比水晶好看一点
4,燃料,fuel,1,10,2000,10,0,0,0,2,普通的燃料，可以将温度提高
5,废料,garbage,0,-2,300,-10,None,1,None,None,加工产生的废料
```
